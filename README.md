
# Programa de inventario de productos

![alt text](<Diagrama de flujo.png>)

## Descripción

Este programa solicita al usuario el nombre, precio y cantidad de un producto.

Luego valida que el precio sea un número decimal y que la cantidad sea un número entero.

Después calcula el costo total multiplicando el precio por la cantidad.

Finalmente muestra toda la información en pantalla de forma clara.

Este proyecto es un programa simple en Python que calcula el costo total de un producto según su precio y cantidad.

El programa es una aplicación básica de consola diseñada para estudiantes que están aprendiendo Python.  
Se utiliza para practicar el uso de entrada de datos, validación, ciclos y operaciones matemáticas.

El sistema solicita al usuario el nombre, el precio y la cantidad de un producto, y luego calcula el costo total.

## Cómo funciona

1. El programa solicita al usuario el nombre de un producto.
2. Después pide el precio del producto y verifica que sea un número decimal válido.
3. Utiliza un ciclo para repetir la solicitud si el valor ingresado no es válido.
4. Luego pide la cantidad del producto y valida que sea un número entero.
5. El programa multiplica el precio por la cantidad para calcular el costo total.
6. Finalmente muestra en pantalla la información del producto y el total calculado.

El programa valida los datos ingresados para evitar errores.  
Asegura que el precio sea un número decimal y que la cantidad sea un número entero.  
El costo total se calcula mediante una operación de multiplicación.

## Cómo ejecutar el algoritmo 
Asegúrate de tener Python instalado (versión 3.x). Puedes verificarlo en la terminal o consola con:

python --version

Descarga el archivo inventario.py o clona el repositorio (ver siguiente sección).

Abre la terminal o consola y navega hasta la carpeta donde está el archivo:

cd ruta/del/proyecto

Ejecuta el programa con:

python inventario.py

(En algunos sistemas puede ser python3 inventario.py).

Interacción con el programa:

Ingresa el nombre del producto cuando te lo pida.

Ingresa el precio (número decimal).

Ingresa la cantidad (número entero).

El programa calculará y mostrará el total automáticamente. 

## Cómo clonar el proyecto 
Abre la terminal o consola.

Navega a la carpeta donde quieres guardar el proyecto, por ejemplo:

cd ruta/donde/quieres/clonar

Ejecuta el comando git clone seguido de la URL del repositorio. Por ejemplo:

git clone https://github.com/tu-usuario/inventario-python.git

Se descargará una carpeta llamada inventario-python (o el nombre de tu repo) con todo el proyecto.

Entra a la carpeta descargada:

cd inventario-python

Ahora puedes ejecutar el programa con:


python inventario.py
